# Text Editor Angular Version

### Run the project

```sh
$ npm  start
```

### Features Inlcuded

* Format text:
    * Bold
    * Underline
    * Tabulation
    * Italic
*   Save text in memory
    * The first time get the text from the service
* Synonyms
    * Get Synonyms from the service
    * Replace original word with the synonyms (keep the styles)
* Modify the original text
